# Deutsch Studio

Site de estudos de alemão (A1.1 → A1.2) — estático, offline, sem backend, versão de teste apenas. --- preview alemao-studio.vercel.app

## Rodar localmente

```
npm install
npm run dev
```

## Voz (Hören / Shadowing / TTS)

O site usa a Web Speech API (`SpeechSynthesis`) do navegador para a pronúncia em alemão.

**Recomendado: Microsoft Edge.** Os modelos de voz de IA (neurais) disponíveis no Edge são melhores que os de outros navegadores — pronúncia mais natural e mais próxima de um falante nativo. Em outros navegadores a qualidade da voz pode ser inferior dependendo das vozes instaladas no sistema.
